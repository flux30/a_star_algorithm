How A* Algorithm Works

A* combines actual cost from start (g(n)) with heuristic estimate to goal (h(n)). The algorithm maintains OPEN and CLOSED lists to track exploration progress.

  • 1

    Initialize Search

    Place start node in OPEN list with g(n)=0 and h(n) estimated

  • 2

    Select Best Node

    Pick node with lowest f(n) = g(n) + h(n) from OPEN list

  • 3

    Expand & Explore

    Move selected node to CLOSED list and explore its neighbors

  • 4

    Reconstruct Path

    When goal found, backtrack through parent pointers to get optimal route
